# Procedural Generation

This repository is an sandbox to investigate different methods of generating levels.

## Cellular Automata Cave Generation

### Algorithm

Using [this page](https://www.roguebasin.com/index.php/Cellular_Automata_Method_for_Generating_Random_Cave-Like_Levels), I was able to implement one of the most primitive
forms of procedural generation. Essentially it randomly fills a grid with empty or filled (in this case `True` or `False`) and then using the number of filled neighbors, determines if it becomes filled or empty. Several passes of this smooth are done and then a border is added with a final smooth pass to round down rough edges.
